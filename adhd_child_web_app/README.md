# 宝宝的情绪小助手

这是一个专门为ADHD儿童设计的心理训练网页应用，通过情景模拟、情绪管理和社交技能训练，帮助孩子更好地控制情绪、改善注意力。

## 功能特色

### 🎯 核心训练模块

1. **注意力训练** - 提升课堂专注能力
2. **情绪管理** - 学会识别和控制情绪
3. **社交技能** - 改善与同伴的互动方式

### 🎭 情景模拟

基于真实校园场景设计的互动情景：

- **💧 水杯被打翻了** - 学习处理冲突的正确方式
- **😔 朋友不和我玩** - 应对被排除的感受
- **😴 上课好无聊** - 提升课堂参与度
- **⚽ 踢球冲突** - 学习礼貌的社交方式

### 🫁 深呼吸练习

- 指导式深呼吸训练
- 4-4-4呼吸法（吸气4秒-保持4秒-呼气4秒）
- 视觉化圆圈引导

### ⭐ 进度与奖励系统

- **星星奖励** - 完成任务获得星星
- **每日进度** - 追踪每天的学习进度
- **成就记录** - 记录个人成长里程碑
- **本地存储** - 自动保存学习进度

## 设计理念

### 基于宝宝的具体情况

应用内容基于以下真实案例设计：

- **课堂表现**：注意力不集中，容易做自己的事情
- **情绪控制**：遇到冲突时容易采取过激行为（如勒脖子、拉扯衣服）
- **社交互动**：对被排除敏感，用不当方式表达不满
- **无聊处理**：上课无聊时会干扰他人

### 循序渐进的训练方法

1. **情景认知** - 帮助孩子识别当前情况
2. **情绪识别** - 学习识别自己的情绪状态
3. **选择训练** - 提供多种应对方式供选择
4. **即时反馈** - 立即告诉孩子选择的后果
5. **正向强化** - 通过奖励鼓励正确选择

## 技术特点

- **响应式设计** - 适配手机、平板和电脑
- **本地存储** - 无需网络，数据本地保存
- **触摸优化** - 适合儿童操作的大按钮设计
- **视觉反馈** - 丰富的动画和颜色提示
- **简单易用** - 直观的界面，无需成人指导

## 使用指南

### 日常使用

1. **每日训练** - 建议每天完成1-2个情景训练
2. **深呼吸练习** - 情绪激动时使用
3. **进度查看** - 观察星星数量和进度条
4. **成就回顾** - 定期回顾已掌握的技能

### 家长指导

- **陪伴练习** - 初期建议陪伴孩子一起使用
- **现实关联** - 将应用中的情景与现实生活联系起来
- **正面鼓励** - 多表扬孩子的进步和努力
- **持续跟踪** - 观察孩子行为改善情况

## 特色功能详解

### 情景选择系统

每个情景都包含：
- **背景描述** - 详细说明发生的情况
- **情绪状态** - 帮助孩子识别当前情绪
- **多重选择** - 2-4种不同的应对方式
- **结果反馈** - 立即显示选择的结果和建议

### 奖励机制

- ⭐ **1-3星** - 基本正确的选择
- ⭐⭐⭐⭐⭐ **5星** - 最佳选择，显示高级情绪管理能力
- **每日完成奖励** - 完成100%进度获得额外5星

### 进度追踪

- **星星累积** - 长期记录获得的总星星数
- **每日进度** - 当天的学习完成百分比
- **情景完成记录** - 记录已完成的情景训练
- **成就列表** - 个人成长里程碑

## 开发说明

### 自定义扩展

可以通过修改 `script.js` 中的 `scenarios` 对象来添加新的情景：

```javascript
const scenarios = {
    newScenario: {
        title: '新情景标题',
        icon: '🎯',
        description: '情景描述...',
        emotion: 'angry',
        choices: [
            {
                text: '选择1',
                feedback: '反馈内容...',
                stars: 3,
                correct: true
            }
        ]
    }
};
```

### 样式定制

主要样式变量在 `styles.css` 中定义：
- 颜色主题：修改渐变色值
- 字体大小：调整根字体大小
- 动画效果：修改 @keyframes 规则
